Авторегистрация водителей

Водители сами заполняют данные в специальной форме приложения и регистрируются в интересующем агрегаторе, тратя при этом не более 15 минут.

Встроенный документооборот

Заполняя заявку на подключение, водитель автоматически заключает с вами договор.

Мгновенные сообщения

В приложении можно настроить автоматическую рассылку с новостями, акциями, пуш-уведомлениями и пр. Информируйте водителей в один клик!

Ваш личный бренд

Разместите ваши контактные данные для обратной связи на специальном экране, дополните логотипом, и ваше личное брендированное приложение для таксопарка готово!

Мгновенные выплаты под вашим контролем

Настройте моментальные выплаты такси водителям , исход из ваших предпочтений — для новых водителей можно ограничить мгновенный вывод, проверенным водителям — дать зеленый свет. При этом сумму вывода и комиссию вы можете настраивать самостоятельно.

Разбивка баз по городам

Есть несколько таксопарков в разных городах? Наше приложение позволит разграничить базы по регионам — каждый водитель будет числиться в вашей базе того города, где работает.
